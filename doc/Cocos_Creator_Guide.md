# Cocos Creator

一个完整的游戏开发解决方案，包含了轻量高效的跨平台游戏引擎，以及能让你更快速开发游戏所需要的各种图形界面工具。 

Cocos Creator 包含游戏引擎、资源管理、场景编辑、游戏预览和发布 



Scene 场景

#### 项目结构

```
ProjectName（项目文件夹）
├──assets		资源文件夹
├──library		资源库
├──local		本地设置
├──packages		扩展插件文件夹
├──settings		项目设置
├──temp			临时文件夹
└──project.json
```
