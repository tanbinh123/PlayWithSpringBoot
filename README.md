# PlayWithSpringBoot

一个学习 SpringBoot & SpringCloud 的 workplace

# Start
> git clone https://github.com/xiaop1ng/PlayWithSpringBoot.git

使用 `Maven` 构建项目，`main` 程序在 `com.xiaoping.App.java`


注：
- jdk 1.8 +
- maven 3.5 +

# Build
> mvn package

`jar` 包输出在 `/target` 目录下
